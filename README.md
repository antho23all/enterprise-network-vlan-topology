# Enterprise Network with VLANs & Multi‑Switch Design
Enterprise Network Project (Packet Tracer + Lucidchart)
This project demonstrates a small enterprise network designed using Cisco Packet Tracer and documented with a Lucidchart network diagram.
The network includes:

2 Routers (R1, R2)

2 Switches (SW1, SW2)

6 PCs across 3 VLANs

Inter‑VLAN routing (Router‑on‑a‑Stick)

Trunking between switches

Logical segmentation using VLANs 10, 20, and 30

This project showcases practical skills used in Network Support, NOC Analyst, and Junior Network Administrator roles.
